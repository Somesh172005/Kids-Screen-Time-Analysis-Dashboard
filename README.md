# Kids Screen Time Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes children's screen time behavior, device usage patterns, and health impacts using over 9,600 records.

## Objectives

* Analyze average daily screen time
* Identify the most used devices
* Measure screen time limit exceedance
* Study health impacts associated with screen usage
* Compare trends across age groups, gender, and location

## Tools Used

* Power BI
* Power Query
* DAX
* Excel / CSV Dataset

## Dashboard Pages

### Overview Dashboard

* Total Kids
* Average Screen Time
* Most Used Device
* Exceeded Percentage
* Screen Time Analysis by Age, Gender and Location

### Device Analysis Dashboard

* Device Usage Distribution
* Device-wise Screen Time
* Device Usage by Age Group
* Device Usage by Location

### Health Impact Dashboard

* Children Affected
* Most Common Impact
* Health Impact Distribution
* Health Impact by Age Group
* Health Impact by Gender

## Key Insights

* Smartphone is the most used device.
* 85.8% of children exceed the recommended screen time limit.
* Teenagers (13–18) have the highest average screen time.
* Poor Sleep is the most common health impact.
